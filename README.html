killthisdog-game

A wack-a-mole version of Pets Broward's KillThisDog.com website.

Live Demo:

http://killthisdog.com/game-preview

Began with CSS Reset.

Tools

  • CSS Reset
  • Sass with Bourbon
  • CodeKit
  • Koala (optional)

Requirements

  • Sound Object
  • CSS Animations
    • Make the pets pop up when children tags inside the hit-targets contains the classname "on"
  • Timer Component
    • A timer that changes in duration depending on dog view-time. Small dogs and cats should appear for less time.
  • Countdown to Begin Game
    • Style the PLAY SCREEN with a countdown as to when the game begins. 3, 2, 1, Go!!! (or even 'Kill')
